# Ticket Booking System

A MySQL-based Ticket Booking System using transactions and seat locking mechanisms.

## Features
- User management
- Show management
- Seat booking
- Concurrency handling
- Row locking using FOR UPDATE
- SKIP LOCKED implementation

## Technologies Used
- MySQL
- SQL Transactions

## Database Tables
- users
- shows
- seats
- bookings
